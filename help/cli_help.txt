usage: ani_yt.py [-h] [--full-help] [-t] [-dir DIRECTORY] [-neu] [-su] [-sr]
                 [-c CHANNEL] [-mc MERGE_CHANNELS [MERGE_CHANNELS ...]]
                 [--mpv-player {auto,default,android,ssh,termux-x11}]
                 [--clear-cache] [--delete-history]
                 [--clear-history {playlist,videos,unwatched}]
                 [--keep-recent KEEP_RECENT] [--delete-bookmark] [-b] [-l]
                 [-v] [-r] [--monitor MONITOR] [--open-app] [--no-open-app]
                 [--return-app] [--no-return-app] [--fullscreen]
                 [--no-fullscreen] [--gestures] [--no-gestures]
                 {source,download,mpv,search,playlist} ...

Note: Options, if provided, will be processed sequentially in the order they
are listed below.

positional arguments:
  {source,download,mpv,search,playlist}
                        Note: To avoid incorrect handling, positional
                        arguments should be placed after all options.
    source              Manage channel source list.
    download            Download video and skip sponsors using SponsorBlock.
    mpv                 Open with MPV.
    search              Search for a playlist.
    playlist            Open playlist from URL

options:
  -h, --help            show this help message and exit
  --full-help           show extended help for all commands and subcommands.

Environment Options:
  -t, --temp            Use temporary folder (incompatible with
                        -dir/--directory).
  -dir, --directory DIRECTORY
                        Specify working directory (incompatible with
                        -t/--temp).
  -neu, --no-extension-update
                        Disable extension update.

Source Update Options:
  -su, --source-update  Quick update command for `source update`
  -sr, --source-rebuild
                        Quick update command for `source rebuild`

Playlist and Channel Options:
  -c, --channel CHANNEL
                        Create or Update Playlist Data from Link, Channel ID,
                        or Channel Handle.
  -mc, --merge-channels MERGE_CHANNELS [MERGE_CHANNELS ...]
                        Merge playlists from multiple channels (URL, Channel
                        ID, or handle).

MPV Player Options:
  --mpv-player {auto,default,android,ssh,termux-x11}
                        MPV player mode.

Cache and History Options:
  --clear-cache         Clear cache.
  --delete-history      Delete history.
  --clear-history {playlist,videos,unwatched}
                        Clear history: 'playlist' to remove old playlists,
                        'videos' to clear videos in old playlists, 'unwatched'
                        to remove only unwatched videos.
  --keep-recent KEEP_RECENT
                        Number of recent playlists to keep when clearing
                        history.
  --delete-bookmark     Delete bookmark.

Browse and View Options:
  -b, --bookmark        Show bookmark.
  -l, --list            Browse all cached playlists.
  -v, --viewed-mode     Browse all videos in cached playlist. Cached playlists
                        will be cleared after playlist selection.
  -r, --resume          View last viewed video.

Termux-X11 options:
  --monitor MONITOR     X server monitor number (default: 1)
  --open-app            Enable auto-opening Termux-X11 app (default: True)
  --no-open-app         Disable auto-opening Termux-X11 app.
  --return-app          Enable auto-return Termux after playback (default:
                        True)
  --no-return-app       Disable auto-return Termux after playback.
  --fullscreen          Enable fullscreen playback (default: True)
  --no-fullscreen       Disable fullscreen playback.
  --gestures            Enable MPV touch/mouse gestures (default: True)
  --no-gestures         Disable MPV touch/mouse gestures.

Subcommand: source
usage: ani_yt.py source [-h] {add,remove,template,update,rebuild} ...

positional arguments:
  {add,remove,template,update,rebuild}
    template            Create placeholder source file.
    update              Update playlist from all saved sources.
    rebuild             Rebuild playlist from all saved sources (clear then
                        update).

options:
  -h, --help            show this help message and exit

Subcommand: download
usage: ani_yt.py download [-h] [-cat CATEGORY] [-m] url

positional arguments:
  url                   Video url.

options:
  -h, --help            show this help message and exit
  -cat, --category CATEGORY
                        See https://wiki.sponsor.ajay.app/w/Types#Category.
  -m, --mpv             Open downloaded video with MPV. Download progress will
                        not be displayed.

Subcommand: mpv
usage: ani_yt.py mpv [-h] input

positional arguments:

              Android.

options:
  -h, --help  show this help message and exit
